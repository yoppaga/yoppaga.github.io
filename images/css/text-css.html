Dies ist eine Demo Seite für CSS Properties die Text formatieren, Teil von Kapitel CSS Properties.

Die große Überschrift ist eine h1

Ein Absatz Text als p

Die Untergordnete Überschrift ist eine h2

Ein Absatz Text als p. Wenns mehrere Absätze sind, dann wird dazwischen ein Abstand gelassen. Eine Leerzeile, sozusagen. Beachten sie die Darstellung des Absatzes.

Noch ein Absatz Text als p. Hier wurde mit Stylesheet nichts umgestellt. Beachten sie die Darstellung des Absatzes.

Und hier der CSS Code nochmal sichtbar gemacht:

h1,h2 {
    font-family: Roboto, sans-serif;
    font-size: 18px;
    font-weight: bold;
    font-style: italic;

    letter-spacing: 0.2ex;  //  Abstand zw. Buchstaben
    text-decoration: underline;  // unterstrichen
    text-shadow: orange 0 -2px;  // Schatten Effekt
  }
  h1 {
    text-transform: uppercase;   // alles Großbuchstaben
  }
  h2 {
    font-variant: small-caps;    // Kapitälchen
  }
